# nk-design_md-craft

ヒアリング → ドラフト生成 → 修正サイクルを通じて `docs/design.md` をインタラクティブに生成するプラグイン。

## スキル

| スキル | 説明 |
|---|---|
| `ts-dmc-create` | オーケストレーター — ヒアリング→ドラフト→修正→書き出しの全ワークフローを実行する |
| `ts-dmc-draft` | タスクスキル — `dmc-drafter` に委譲して12セクションのドラフトを生成する |

## エージェント

| エージェント | 説明 |
|---|---|
| `dmc-drafter` | リファレンスガイドを読み込み、12セクションの design.md ドラフトを生成する |

## 使い方

`docs/design.md` を作成したいときに `ts-dmc-create` を呼び出す:

- "design.mdを作って"
- "デザインドキュメントを作成して"
- "design document を生成して"

## 前提条件

サブエージェントが `docs/tmp/design-md/design-md.md` をリファレンスガイドとして参照する。スキルを呼び出す前にプロジェクト内に配置しておくこと。
